### 个人分析

由于缺少关键的 DNA 物证, 未来估计除了测谎机的进步, 这个案子似乎已经无解了。

- 班长或其它班干部有经常不参加跑操的习惯或是特权。

  > 同时也说明了跑操并不受欢迎, 大家能躲就躲

- 律师始终围绕着 “强奸” 来进行翻案，一份有瑕疵、粗糙、满是 BUG 的卷宗，真就是一块肥肉。

- 很好奇, 王/杨二人只举证了张志超和王广超的出现, 为什么警方最后仅认定张志超是凶手？

  > 警方若是能公布更多案宗就好了, 视频截图能获得的信息太少了
